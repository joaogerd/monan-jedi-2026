
# MONAN-JEDI 2026

Repositório oficial do Plano Técnico 2026 para implementação do sistema JEDI + MONAN no CPTEC.

## Estrutura Organizacional

O projeto está organizado em seis domínios técnicos paralelos:

- Infraestrutura & Spack
- Modelo (MPAS/MONAN)
- Observações
- Núcleo Variacional (JEDI)
- Avaliação & Métricas
- Documentação & Reprodutibilidade

## Princípios de Execução

- Progressão por marcos verificáveis
- Execução paralela coordenada
- Critério objetivo de aceite
- Reprodutibilidade institucional
- Ausência de ajustes ad hoc

## Milestones 2026

- M1 – Consolidação de Ambiente
- M2 – Baseline Determinístico MPAS
- M3 – Integração 3DVar FGAT
- M4 – Avaliação Formal MPAS-JEDI
- M5 – Transição Controlada MPAS → MONAN
- M6 – Consolidação MONAN-JEDI
- M7 – Hybrid 3DVar – Prova de Conceito

## Fluxo Operacional do Projeto

As atividades seguem o seguinte fluxo formal:

Todo → Pronto para Execução → Em Execução → Em Revisão Técnica → Em Validação → Concluído

Nenhuma tarefa é considerada finalizada sem:

- Evidência técnica documentada
- Critério de aceite cumprido
- Validação cruzada quando aplicável
